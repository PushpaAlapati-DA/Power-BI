Project Description:

This Power BI dashboard presents a comparative analysis of employee data from India and New Zealand using an HR dataset. It visualises key metrics such as headcount, average salary, department distribution, and gender breakdown to identify workforce trends and regional differences. The dashboard enables data-driven HR insights to support strategic workforce planning and decision-making.
